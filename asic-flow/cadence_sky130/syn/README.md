# Synthesis Output Directories

These directories hold the artifacts generated during logic synthesis (Genus).
They are intentionally kept empty in version control — their contents are produced
at run time and are not tracked.

- **`work/`** — intermediate databases and working files created during synthesis.
- **`outputs/`** — final synthesis deliverables (netlist, SDC, MMMC, SDF, scandef).
- **`reports/`** — timing, area, and power reports generated at each stage.
- **`logs/`** — tool execution logs.

All files under these directories are regenerated by running the flow and are
excluded from the repository via `.gitignore`.
